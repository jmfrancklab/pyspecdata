Bruker cw ESR Data¶

Load cw Bruker ESR data, both 1D and 2D.

Check out the Simple UV example to understand how pySpecData locates the file here.

from numpy import *
import pylab as plt
from pyspecdata import *

d.chunk_auto('harmonic','phase')
